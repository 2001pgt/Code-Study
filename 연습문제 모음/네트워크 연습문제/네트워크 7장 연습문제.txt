B. 802.2는 LLC, 802.3~802.22는 MAC

D. 802.11은 LLC, 802.15는 MAC

D. 전송률 향상

B. 버퍼 크기 관리

11. HDLC의 특징으로 옳은 것은? A -> C
A. 비동기식 전송
B. 슬롯 기반 전송
C. 비트 기반 동기식 전송
D. 전파 지연 방식
해설: HDLC는 동기식 전송, ATM,PPP등이 비동기 전송

14. HDLC에서 슬라이딩 윈도우 방식의 흐름제어를 사용하는 ARQ 방식이 아닌 것은? C -> B
A. Go-Back-N 
B. Stop-and-Wait
C. Selective Repeat
D. 모두 사용함
해설: HDLC 는 슬라이딩 윈도우 프로토콜을 사용하며 Go-Back-N ,Selective Repeat 모두 지원

19. Length/Type 필드가 1500보다 작을 경우 해당 필드는 무엇을 의미하는가?   -> B
A. 프로토콜 타입
B. 데이터 길이
C. MAC 주소
D. CRC 코드
해설: Length/Type 필드는 데이터 1500바이트 이상이면 데이터의 길이를 나타내고
이하면 TYPE 을 나타낸다.

20. 이더넷에서 상위 계층에서 받은 데이터가 46바이트 미만일 경우 하는 조치는?   -> C
A. 삭제
B. 압축
C. 패딩 추가
D. 패킷 분할
해설: 이터넷 프레임의 최소크기는 64 바이트이기 때문에 패딩을 추가하여 데이터를 64바이트로 맞춘다.
