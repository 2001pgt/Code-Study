네트워크 연습문제 9장

1. 네트워크 계층에서 데이터를 목적지까지 전달할 때 사용하는 단위는?
A. 세그먼트
B. 프레임
C. 셀
D. 패킷

2. 다음 중 네트워크 계층에서 가장 핵심적인 역할은?
A. 응용 서비스 제공
B. 데이터 압축
C. 패킷 전달
D. 전송 속도 보장

3. 라우팅 알고리즘의 주된 목적은 무엇인가?
A. 오류 제어
B. 가장 빠른 경로 선택
C. 암호화 처리
D. 흐름 제어

4. 인터넷에서 LAN과 LAN을 연결하는 장치는?
A. 스위치
B. 게이트웨이
C. 라우터
D. 허브

5. IP 프로토콜의 특징으로 가장 적절한 설명은?
A. 패킷 도착 순서를 보장
B. 항상 동일한 경로 사용
C. 최적의 경로로 패킷 전달 시도
D. 에러 자동 수정

6. 가상회선(virtual circuit)의 특징으로 옳은 것은?
A. 순서 보장 불가
B. 셋업 단계 없음
C. 정해진 경로를 사용
D. 패킷 손실 발생 가능

7. 다음 중 IP 주소의 길이로 올바른 것은?
A. 2바이트
B. 4바이트
C. 6바이트
D. 8바이트

8. IPv4 주소에서 4번째 바이트가 의미하는 것은?
A. 서브넷 마스크
B. 국가 코드
C. 네트워크 번호
D. 호스트 주소

9. 다음 중 C 클래스 IP 주소 체계의 구성 방식은?
A. 1바이트 네트워크 + 3바이트 호스트
B. 2바이트 네트워크 + 2바이트 호스트
C. 3바이트 네트워크 + 1바이트 호스트
D. 4바이트 호스트 주소

10. IP 주소의 D 클래스는 무엇을 위해 사용되는가?
A. 실시간 전송
B. 루프백 주소
C. 멀티캐스트
D. VPN 설정

11. 루프백(loopback) 주소는 어떤 용도인가?
A. 브로드캐스트 전송
B. 내부 자기 자신 테스트
C. 네트워크 연결 확인
D. DNS 서버 확인

12. 도메인 이름의 역할은?
A. MAC 주소 암호화
B. IP 주소를 외부에서 숨김
C. IP 주소를 사람이 기억하기 쉬운 이름으로 표현
D. DNS 캐시를 초기화

13. DNS의 기능은 무엇인가?
A. 데이터 전송량 제한
B. 도메인 → IP 주소 변환
C. 라우팅 경로 설정
D. 네트워크 트래픽 제어

14. DNS 서버가 없는 경우 어떤 일이 발생할 수 있는가?
A. 패킷 손실
B. 인터넷 접속 불가
C. DNS 자동 복구
D. MAC 주소 충돌

15. DNS 계층 구조에서 가장 상위에 있는 서버는?
A. ISP DNS
B. 루트 DNS
C. 지역 DNS
D. 사용자 DNS

16. 자주 방문하지 않은 사이트를 처음 접속할 때 웹브라우저가 느려지는 이유는?
A. 서버 부하
B. DNS 루트 서버에 직접 요청
C. 바이러스 감지
D. IP 충돌

17. 서브넷 마스크의 목적은?
A. DNS 이름 검색
B. 네트워크와 호스트 주소 분리
C. 패킷 암호화
D. 포트 번호 설정

18. 다음 중 A 클래스의 서브넷 마스크로 옳은 것은?
A. 255.0.0.0
B. 255.255.0.0
C. 255.255.255.0
D. 0.0.0.255

19. 서브넷 마스크 연산 방식은 어떤 논리 연산을 사용하는가?
A. OR
B. XOR
C. AND
D. NAND
